# Dashboard Horux · Instrucciones

## Subir a GitHub (HoruxHH)
1. Entra a https://github.com/horuxbc/HoruxHH
2. Click en `dashboard.html`
3. Click en lápiz (Edit)
4. Selecciona todo (Ctrl+A) y borra
5. Pega el nuevo
6. Commit changes
7. Espera 90 seg → https://horuxbc.github.io/HoruxHH/dashboard.html

## Filtros del dashboard
Solo cuenta vacantes verdaderamente activas (estados Activa, Terna, Propuesta).
NO incluye: Facturadas, En garantía, On Hold.

Para ver el consolidado completo (incluyendo on hold y garantía), ir a sección Vacantes.
